# b4geoip

b4geoip публикует набор `sing-box` `.srs` файлов.

Доступные категории:

- geoip:blizzard
- geoip:epicgames
- geoip:sony
- geoip:valve

Примеры прямых ссылок на последние версии (обновляются раз в день):

```text
https://github.com/TakaiSaisei/b4geoip/releases/latest/download/blizzard.srs
https://github.com/TakaiSaisei/b4geoip/releases/latest/download/sony.srs
```
